
  # CNC_Frontend

  This is a code bundle for B2B Industrial Website. The original project is available at https://www.figma.com/design/MLvRfSRUiFjGJxd7AXbDjo/B2B-Industrial-Website.

  ## Running the code (Next.js)

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  Run `npm run build` to build production bundle.

  Run `npm run start` to run the production server.
